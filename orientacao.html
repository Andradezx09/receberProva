Orientações para o envio da prova

  1. Acesse o seu GitHub e copie o link do repositório onde está armazenada a sua prova.
  2. No VSCode, crie um novo arquivo utilizando o seguinte formato para o nome: link_nomeSobrenome_turmaXX.html
  3. No conteúdo deste arquivo:
    1. Na primeira linha, informe se realizou a prova em sala ou em casa:
      1. Escreva SIM se realizou a prova em sala.
      2. Escreva NÃO se realizou a prova em casa.
    2. Na segunda linha, coloque o link da sua prova que esta no GitHub.
  4. Em seguida, crie um novo arquivo no VSCode com o seguinte formato de nome: nomeSobrenome_turmaXX.html
  5. No conteúdo deste segundo arquivo, inclua o código correspondente à sua prova, que se encontra no seu repositório no GitHub.

Processo para enviar os arquivos:

  1. Verifique os nomes dos arquivos:
    • Certifique-se de que os arquivos criados estão com os nomes corretos.
  2. Publique apenas os dois arquivos solicitados:
    • Não inclua outros arquivos além dos especificados nas instruções.
    • Vá para o GitHub DeskTop
      • Clique em repositório
        • Selecione "Repositório Settings"
        • Selecione "Git config"
        • Selecione "Local Git Config"
        • Preencha com os seus cadastrados no GitHub os campos Nome e E-mail.
        • Clique em salvar
      • Clique em File
        • Clique em "Option"
          • Verifique se é a sua conta que esta conectada no GitHub.
          • Caso contrário, desconecte a conta conectada e coloque a sua.
          • Clique em "salvar".
    • Vá para o VSCode
      • Selecione o icone: "Controle de Código Fonte"
        • Escreva no campo Mensagem o seu Nome, sobrenome e turma
        • clique no "+" na frente de cada arquivo que será enviado.
        • clique em "Confirmação"
      • Durante o processo de publicação do arquivo, será exibida a opção FORK.
        • Certifique-se de selecionar a opção azul indicada na imagem.
